\newcommand{\prodpermin}[1]{ % #1 decimal number to be rounded to 3 digits
    \DIVIDE{60}{#1}{\sol}
    \num[round-mode=places,round-precision=3]{\sol}%
}

			row sep=-2pt,
			column sep=-1mm,
			matrix of nodes,
			ampersand replacement=\&] (#2) (output) at (0,{\stratalistmath}) {

	\end{tikzpicture}};;
}

\newcounter{strata}
\newcommand{\consstratalist}[1]{
    \let\matrixcontents\empty
    \setcounter{strata}{0}
    \foreach \l/\r in {#1}{
        \expandafter\gappto\expandafter\matrixcontents\expandafter{\expandafter\smallicon\expandafter{\l} \&}
        \expandafter\gappto\expandafter\matrixcontents\expandafter{\r \\}
        \stepcounter{strata}
    }
}

\section{Electrification \& Symbols}

Almost all buildings in the \textbf{Old World}, excluding \textit{farms} and \textit{Lumberjacks}, can be electrified for up to \textit{200\%} productivity.
\underline{Buildings marked with\smallicon{Electricity}require electricity by default}.\\
\\
Some buildings have variants with different statistics, these are markes with \oldworld{}Old World, \newworld{} New World, \arctic{} Artic and \enbesa{} Enbesa.
